## Тестовые сценарии

### 1. Авторизация как NetManager сети Barbos

```bash
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "77012345001",
    "password": "password123"
  }'
```

**Доступные действия:**
- Просмотр всех работников сети `BARBOS_KZ`
- Просмотр работников конкретных точек сети

### 2. Получение работников сети

```bash
# Все работники сети Barbos (staff + manager)
curl -X GET http://localhost:8080/api/v1/networks/BARBOS_KZ/staff \
  -H "Authorization: Bearer YOUR_TOKEN"

# Только барберы (staff)
curl -X GET http://localhost:8080/api/v1/networks/BARBOS_KZ/staff?roles=staff \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 3. Получение работников точки

```bash
# Все работники точки
curl -X GET http://localhost:8080/api/v1/points/BARBOS_ALMATY_ESENTAI/staff \
  -H "Authorization: Bearer YOUR_TOKEN"

# Только барберы
curl -X GET http://localhost:8080/api/v1/points/BARBOS_ALMATY_ESENTAI/staff?roles=staff \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 4. Авторизация как Manager точки

```bash
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "77012345101",
    "password": "password123"
  }'
```

**Ограничения:**
- Может видеть только работников своей точки `BARBOS_ALMATY_ESENTAI`
- Не может просматривать работников других точек

### 5. Получение услуг мастера

```bash
# Получить все услуги конкретного мастера
curl -X GET http://localhost:8080/api/v1/masters/77012345111/services \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Примерный ответ:**
```json
{
  "services": [
    {
      "id": "uuid-here",
      "name": "Мужская классическая стрижка",
      "price": 8000.00,
      "duration_minutes": 40,
      "active": true
    },
    {
      "id": "uuid-here",
      "name": "Модельная стрижка",
      "price": 10000.00,
      "duration_minutes": 50,
      "active": true
    }
  ]
}
```

### 6. Получение мастеров по услуге

```bash
# Найти всех мастеров, которые оказывают конкретную услугу
curl -X GET http://localhost:8080/api/v1/services/{service_id}/masters \
  -H "Authorization: Bearer YOUR_TOKEN"
```
