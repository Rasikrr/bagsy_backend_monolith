Добавление мастера
ФИО, телефон, Роль, код точки, тут тоже чек бокс can_provide_services. На указ номер через greenApi приходит ссылка вида app.bagsy.kz/invite/short_code. по этому коду я кидаю запрос на получение
{
"invite_valid": true,
"network_name": "Сеть Салонов Beauty",
"point_name": "Филиал на Абая",
"role": "master",
"can_provide_services": true,
"user_exists": true,
"already_linked": false,
"current_point_name": null, // если already_linked = true, то название текущей точки
"expires_at": "2024-02-05T10:00:00Z"
}
